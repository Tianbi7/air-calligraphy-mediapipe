# 空气书画：基于MediaPipe手部与姿态跟踪的实时毛笔水墨渲染系统
SIGGRAPH 交互式演示工程 | 实时体感空中书法交互平台

## 项目摘要
本项目实现一套摄像头驱动的实时体感空气书法仿真系统，适用于SIGGRAPH海报现场交互演示。依托MediaPipe手部、全身姿态关键点跟踪算法，捕捉指尖运动轨迹，生成具备粗细渐变、墨色晕染、宣纸纹理的传统楷书毛笔笔迹。用户可通过简易手势切换笔锋粗细、更换墨色，系统内置自动清屏、作品自动保存功能；同时附带全身人体姿态分析可视化模块，可用于人机交互相关可视化展示。整套工程轻量化、模块化拆分，便于现场部署与代码复现。

## 核心技术功能
### 1. 双指毛笔笔锋控制
- 食指书写：细笔楷书，笔画起笔收笔带自然笔锋渐变
- 拇指书写：粗笔浓墨，适合大字、厚重书法效果

### 2. 传统水墨真实渲染
- 带随机杂色纹理仿古宣纸底色
- 运动速度自适应笔画粗细：移动越慢笔触越粗，快速划过笔触纤细
- 分段渐变透明度+高斯模糊，模拟墨水在宣纸上自然晕开效果

### 3. 无接触手势交互逻辑
1. 双手完全张开保持2秒：循环切换三种墨色（浓黑墨/淡灰墨/朱砂红墨）
2. 5秒无书写动作：自动清空宣纸画布
3. 留有笔迹闲置5秒：自动将当前书法作品保存至本地输出文件夹

### 4. 全身人体姿态实时解析
实时检测人体距离、身体前倾角度、双手开合状态，界面可视化输出姿态数据，可用于人机交互行为分析展示。

### 5. 模块化解耦设计（适配学术开源规范）
渲染算法、跟踪检测器、工具函数、全局参数完全分离，审稿人可单独提取水墨渲染模块用于论文效果复现。

## 环境依赖安装
```bash
pip install opencv-python mediapipe numpy
```

## 运行前置准备
1. 将 `hand_landmarker.task`、`pose_landmarker_lite.task` 两个模型文件放入 `models` 文件夹
2. 进入代码目录 `src` 执行主程序
```bash
cd src
python main.py
```

## 操作交互说明
1. 伸出食指：细笔楷书书写
2. 伸出拇指：粗笔浓墨书写
3. 双手同时完全张开2秒：切换墨色
4. 5秒无手部动作自动清空画布
5. 留有笔迹闲置5秒自动保存作品至 `output_calligraphy`
6. 按下键盘 Q 键关闭程序
